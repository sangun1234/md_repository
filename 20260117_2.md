## description 명령어
- 인터페이스에 설명을 추가하는 명령어
- 예: description [설명 내용]
- 인터페이스 상태 확인 시 설명이 함께 표시되어 관리에 도움

```ios
        Router> enable
        Router# configure terminal
        Router(config)# interface GigabitEthernet0/1
        Router(config-if)# description connected to Server A
        Router(config-if)# exit
        Router(config)# exit
        Router# show interfaces GigabitEthernet0/1 description
```
- 출력 예시:
```plaintext
Interface              Status         Protocol Description
GigabitEthernet0/1    up             up       connected to Server A
```

### show interfaces
- 인터페이스의 상태와 통계를 확인하는 명령어


### show ip interfaces
- 인터페이스의 IP 관련 정보를 확인하는 명령어



## L2스위치에 IP주소 할당하기
- 스위치에 IP주소를 할당하여 원격 관리 가능
```ios
        Switch> enable
        Switch# configure terminal
        Switch(config)# interface vlan 1
        Switch(config-if)# ip address 192.168.1.2
        Switch(config-if)# no shutdown
        Switch(config-if)# exit
        Switch(config)# exit
        Switch# show ip interface brief
```
- 출력 예시:
```plaintext
Interface              IP-Address      OK? Method Status                Protocol
Vlan1                 192.168.1.2     YES manual up                    up
```

## SVI (Switched Virtual Interface)
- VLAN에 할당된 가상 인터페이스
- 스위치가 L3 기능을 수행할 수 있도록 함
- SVI를 통해 스위치에 IP주소를 할당하고 원격 관리 가능
- SVI는 VLAN마다 하나씩 생성 가능
- 예: VLAN 10에 SVI 생성 및 IP주소 할당
```ios
        Switch> enable
        Switch# configure terminal
        Switch(config)# vlan 10
        Switch(config-vlan)# exit
        Switch(config)# interface vlan 10
        Switch(config-if)# ip address 192.168.10.1 255.255.255.0
        Switch(config-if)# no shutdown
        Switch(config-if)# exit
        Switch(config)# exit
```
